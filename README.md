# spectralAnalysis
## About The Project
Scripts for Graphing Spectral Data Using Standard TOPAS PHSP Phase-Space Data Files
## Getting Started
### Installation
git clone 'SSH-Key'
## Usage
### reader.py
Spectral Data Reading Library
### plotting.py
Spectral Data Plots Library
### sphere.py
Spectral Data Sphere Measurement Analysis Library
